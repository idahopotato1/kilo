# kilo

A 2D game library written in TypeScript.

## Development

Install dependencies:
```bash
npm install
```

Run tests:
```bash
npm test
```
Run tests with coverage reporting:
```bash
npm run test:cov
```

Update docs:
```bash
npm run docs
```
To serve docs locally (at localhost:5000):
```bash
npm run docs:serve
```
